# Color Tools

## Премущества
- Наличие трех инструментов
- Интуитивный и понятный интерфейс
- Сохранение и копирование цветовой палитры

## Инструменты
- **Выбор цвета**: определение цвета через цветовой круг и квадрат
- **Палитра**: Выбор цвета через градиент. Поддержка HSL, HSV
- **Подбор цветов**: Наличие круга с настройкой градуса, видом точек и подбором градиента с проверкой этого градиента в сайте примере.
